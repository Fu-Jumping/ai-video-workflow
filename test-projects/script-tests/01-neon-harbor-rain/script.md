# Neon Harbor Rain

## 类型

近未来科幻悬疑短片

## 一句话故事

一个负责清理城市记忆缓存的夜班技师，在暴雨中的港口服务器站发现一段本应被删除的童年记忆，而记忆里的孩子正站在她身后。

## 主要人物

- 林澈：32 岁，城市记忆缓存清理员，冷静、克制，习惯把所有情绪压进工作流程。
- 阿雾：12 岁左右，出现在废弃港口的孩子，知道很多不该知道的事。
- 周栖：远程调度员，只通过耳机出现，语气轻松但不断催促林澈完成删除。

## 剧本

### 场景 1：外景，霓虹港口，夜，暴雨

雨像细密的电流落在港口地面。废弃吊臂挂着残破广告屏，屏幕上循环闪烁一句标语：把过去交给系统，把明天留给自己。

林澈穿着深色防水外套，拖着银色工具箱穿过积水。她经过一排关闭的记忆回收柜，每个柜门后都有微弱蓝光。

耳机里传来周栖的声音。

周栖：十九号缓存站离线三天了。总部要你做一次硬删除，十分钟内结束。

林澈：硬删除之前要做完整校验。

周栖：你总是这么爱惜别人的过去。

林澈没有接话。她刷卡进入服务器站。

### 场景 2：内景，港口服务器站，夜

服务器站里没有灯，只有机柜内部的蓝色光带。雨水沿着天花板裂缝滴落，砸在金属地板上。

林澈打开终端，屏幕显示：缓存对象 7139，删除失败，权限来源未知。

她输入诊断命令。服务器弹出一段影像：一个小女孩坐在旧厨房桌前，面前是一只碎裂的白瓷碗。女孩抬头，正对镜头。

女孩：林澈，别删我。

林澈的手停住。

周栖：你那边怎么停了？

林澈：缓存里有我的名字。

周栖沉默半秒。

周栖：巧合。城市里叫这个名字的人很多。

林澈：她说话时看着我。

周栖：缓存会根据观看者生成回声。你知道规程。

林澈关掉耳机降噪，站在机柜间听。服务器深处传来轻轻的敲击声。

### 场景 3：内景，服务器站走廊，夜

林澈拿着手电沿着狭窄走廊前进。墙面有很多被水泡开的儿童涂鸦贴纸，像是有人把旧学校搬进了机器里。

尽头坐着阿雾。她穿着干净的校服，脚边没有水迹。

阿雾：你来晚了。

林澈：你是谁？

阿雾：你把我存进来的时候，说十年以后一定会来接我。

林澈：我没有存过任何私人记忆。

阿雾：因为你把自己也删掉了一部分。

林澈靠近，手电光照到阿雾身后的机柜。机柜门内不是电路，而是一间老厨房的投影。桌上放着碎裂的白瓷碗。

林澈低声说：这是我母亲的厨房。

阿雾：那天你砸了碗。她没有骂你。第二天她就不记得你了。

林澈眼神动摇。

### 场景 4：内景，服务器站主控室，夜

周栖强制接入主控屏。

周栖：林澈，听我说。那个缓存不是私人备份，是早期城市实验残留。里面有很多人的童年片段。它会用你的记忆让你放弃删除。

林澈：为什么总部这么急？

周栖：因为它开始反向写入了。它不只保存记忆，它在制造身份。

阿雾站在林澈身后，看着屏幕。

阿雾：我不是制造出来的。我只是还没有被允许活完。

周栖：不要和它对话。

林澈看向硬删除按钮。按钮旁边还有一个灰色选项：迁移到本地隔离区。

周栖：林澈，隔离区会把责任落到你个人名下。

林澈：如果它只是诱导我，它为什么知道我不敢负责？

她拔掉总部远程线缆。警报响起。

### 场景 5：外景，霓虹港口，黎明前

暴雨变小。港口天边有一线灰白。

林澈抱着工具箱走出服务器站。阿雾跟在她身边，身体在雨里有轻微闪烁。

阿雾：你真的要带我走？

林澈：我只答应把你带到能回答问题的地方。

阿雾：那如果答案证明我不是真的呢？

林澈停下，看着积水里两个人的倒影。倒影中阿雾的脸一瞬间变成小时候的林澈。

林澈：那也要由你自己听见。

远处城市广播响起：十九号缓存站已完成删除。

林澈关掉工具箱上的信号灯，带着阿雾走进尚未亮起的街道。

## 结尾画面

港口广告屏重新亮起，标语出现错误：把过去交给自己，把明天留给系统。
