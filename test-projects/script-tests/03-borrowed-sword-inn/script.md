# Borrowed Sword Inn

## 类型

古风武侠喜剧短片

## 一句话故事

一个不会武功的客栈账房为了讨回被欠的房钱，临时借来一把名剑假扮高手，却意外卷入两路江湖人争夺密函的混乱夜晚。

## 主要人物

- 唐小满：24 岁，客栈账房，算盘打得快，胆子不大，嘴上不肯输。
- 宋青衣：真正的剑客，外表冷淡，剑被唐小满误拿。
- 胡掌柜：客栈老板，爱钱如命，遇事先躲后算账。
- 赤衣客：江湖跑腿人，带着密函，过度自信。
- 白面公子：追密函的人，讲究排场，武功一般。

## 剧本

### 场景 1：内景，三更客栈大堂，黄昏

客栈大堂冷冷清清。唐小满坐在柜台后拨算盘，越拨越生气。

唐小满：三号房欠两晚，五号房摔一只碗，七号房说自己是侠客所以不付茶钱。江湖人是不是都不识数？

胡掌柜从柜台下探出头。

胡掌柜：声音小些。七号房那位佩剑。

唐小满：佩剑怎么了？佩剑能抵房钱？

话音刚落，一把长剑从楼梯上滚下来，正好停在唐小满脚边。剑鞘乌黑，银色剑穗干净利落。

胡掌柜：能。

唐小满捡起剑，发现剑柄上刻着两个字：青衣。

### 场景 2：内景，客栈二楼走廊，夜

唐小满提着剑去七号房讨账。她还没敲门，门先开了。赤衣客冲出来，把一封密函塞进她怀里。

赤衣客：宋青衣，东西先交你，我引开他们。

唐小满：我不是宋青衣。

赤衣客看见她手里的剑，肃然起敬。

赤衣客：高手都这么说。

他翻窗跑了。

唐小满低头看密函，又看剑。

走廊另一头，白面公子带着两名随从出现。

白面公子：宋女侠，把信交出来。

唐小满把密函藏进账本。

唐小满：先把房钱交出来。

白面公子愣住。

白面公子：什么？

唐小满：江湖规矩我不懂，客栈规矩先付账。

### 场景 3：内景，客栈大堂，夜

所有人围在大堂。唐小满站在桌上，努力装得从容。宋青衣坐在角落，发现自己的剑在唐小满手里，没有立刻出声。

白面公子：既然宋女侠不肯交信，就别怪我请教剑法。

唐小满小声问胡掌柜。

唐小满：请教剑法要不要收费？

胡掌柜：如果你活下来，可以收场地费。

白面公子拔剑冲来。唐小满吓得闭眼，手里的剑鞘刚好卡住桌腿。她一转身，整张桌子被带起来，砸翻两个随从。

众人沉默。

胡掌柜立刻鼓掌。

胡掌柜：好一招桌转乾坤。

唐小满睁眼，赶紧摆出高深表情。

唐小满：看懂了？

白面公子后退半步。

白面公子：没看懂。

唐小满：没看懂就对了。

### 场景 4：内景，客栈厨房，夜

唐小满趁乱躲进厨房，宋青衣跟进来。

宋青衣：剑是我的。

唐小满吓得差点把锅盖扣到自己头上。

唐小满：你才是宋青衣？

宋青衣点头。

唐小满：那你刚才为什么不说？

宋青衣：我想看你怎么收场。

唐小满：现在看完了，评价如何？

宋青衣：账房不错，剑法没有。

外面传来白面公子的声音：搜厨房。

唐小满把密函递给宋青衣。

唐小满：东西给你。顺便把七号房的房钱也给了。

宋青衣接过密函，从袖中取出一锭银子。

唐小满：这么多？

宋青衣：包括你刚才替我挨打的钱。

唐小满立刻把银子收好。

唐小满：女侠请放心，我还能再挨一会儿。

### 场景 5：内景，客栈大堂，深夜

宋青衣终于出剑。她动作极快，只用剑鞘点落白面公子的剑，又把赤衣客从窗外踢回大堂。

赤衣客摔在地上，还不忘整理发带。

赤衣客：宋女侠，信可还安全？

唐小满从柜台后探头。

唐小满：信安全，账也安全。你欠一扇窗。

白面公子不服。

白面公子：那她呢？她冒充高手，怎么算？

唐小满举起账本。

唐小满：我没有冒充。我从头到尾只说先付账。是你们自己误会。

宋青衣看了她一眼，嘴角很轻地弯了一下。

宋青衣：确实。她的算盘，比剑准。

### 场景 6：内景，客栈大堂，清晨

江湖人走光了。大堂一片狼藉。胡掌柜拿着损坏清单，眼含热泪。

胡掌柜：小满，我们发财了。

唐小满把一块新牌子挂到柜台前：江湖纠纷，先交押金。

宋青衣准备离开，把剑重新挂回腰间。

唐小满：女侠下次还住七号房？

宋青衣：你不怕麻烦？

唐小满：怕。但你付钱爽快。

宋青衣丢下一枚铜钱。

宋青衣：订金。

唐小满接住铜钱，笑得很克制。

唐小满：七号房给你留着。窗户另算。

## 结尾画面

客栈门口新挂的牌子在晨风里晃动：三更客栈，欢迎大侠，谢绝赊账。
