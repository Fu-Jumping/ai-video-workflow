# Blue Breakfast Van

## 类型

当代现实治愈家庭公路短片

## 一句话故事

一个离家多年的女儿开着母亲留下的蓝色早餐车，沿着旧送餐路线寻找最后一位欠账客人，最终发现那不是债务，而是母亲留给她的告别方式。

## 主要人物

- 许南：29 岁，城市广告剪辑师，刚辞职回乡，和母亲长期疏离。
- 许明月：许南的母亲，已故，只通过账本、录音和他人口述出现。
- 老吴：修车铺老板，沉默寡言，帮许南修早餐车。
- 小禾：高中生，每天替外婆买早餐，话多直接。

## 剧本

### 场景 1：外景，县城修车铺，清晨

一辆褪色的蓝色早餐车停在卷帘门前，车身写着“明月早餐”。字迹被太阳晒得发白。

许南站在车旁，手里拿着一串旧钥匙。老吴趴在车底敲了两下。

老吴：还能开。就是你妈舍不得换零件，哪里都响。

许南：卖掉之前能开到镇上就行。

老吴钻出来，看了她一眼。

老吴：她以前一天跑四个村，没少靠这车养你。

许南：我知道。

老吴：你不知道。知道和听说，不是一回事。

许南没有反驳。她打开车门，驾驶座上放着一本油渍斑斑的账本。

### 场景 2：内景，早餐车，清晨

许南翻账本。每页都是包子、豆浆、茶叶蛋的小账。最后一页写着：槐树坡，欠 18 元，必须亲自送。

旁边夹着一张便签：南南，如果你看到这里，说明车还没卖。

许南盯着便签很久，把账本合上。

小禾敲车窗。

小禾：今天还卖不卖？我外婆说你妈家的豆浆不兑水。

许南：不卖。

小禾：那你开火干吗？

许南低头，发现自己已经按下了蒸箱开关。白汽从车内升起。

### 场景 3：外景，乡间公路，上午

蓝色早餐车沿着窄路前进。车内电台信号断断续续，放着十几年前的本地天气预报。

许南按照账本路线停在第一个村口。一个卖菜阿姨认出车，笑着跑过来。

卖菜阿姨：明月今天怎么让你来了？

许南沉默。

阿姨的笑慢慢收住。

许南：她走了。

阿姨低头，从布袋里拿出一枚硬币放在车窗边。

卖菜阿姨：那她肯定没告诉你，她每年冬天都给我们这几个摊送热豆浆，不收钱。说你小时候怕冷，她见不得小孩冷。

许南看着硬币，不知道该不该收。

### 场景 4：外景，槐树坡村口，午后

槐树坡只有一棵很大的老槐树和几间低矮房子。小禾坐在车副驾驶，嚼着许南蒸坏的馒头。

小禾：你这馒头硬得像试卷。

许南：没人让你吃。

小禾：我外婆说，难吃也要鼓励年轻人。

许南按账本找到最后一户。门开着，屋里没有人。桌上放着一台老式录音机，贴着纸条：给许南。

许南按下播放。

许明月的声音响起，带着一点电流噪声。

许明月：南南，槐树坡没有欠账客人。十八元是你第一次帮我卖早餐那天算错的钱。你哭了一下午，说以后再也不做生意。

许南慢慢坐下。

许明月：我没有怪你。后来你走得远，我也没怪你。只是有些话当面说，你肯定嫌我啰嗦。

录音里有轻轻的笑声。

许明月：这条路你要是跑完了，就知道我不是一个人在这里过日子。你不用替我难过。

### 场景 5：外景，槐树坡老槐树下，傍晚

许南把早餐车停在树下，打开蒸箱。蒸汽升起来，村里的老人和孩子慢慢围过来。

她第一次认真学着母亲的手势盛豆浆。小禾站在一旁负责收钱，收得乱七八糟。

老吴开着摩托赶来，把新换的车标递给许南。车标还是蓝色，只是字重新喷过。

老吴：你妈说，如果你想卖车，就让我别拦。如果你想留下，就把这个给你。

许南看着车标。上面写着：南南早餐。

她摇头，把车标翻过来，用马克笔写下：明月和南南早餐。

小禾：名字太长了，招牌放不下。

许南：那就写小一点。

### 场景 6：外景，县城街口，第二天清晨

蓝色早餐车停在熟悉的街角。许南笨拙地摆出豆浆和茶叶蛋。

第一位客人走来。

客人：老板，豆浆甜的咸的？

许南一愣，低头看母亲账本。账本最后一页已经没有欠账，只夹着一张新纸。

她在纸上写下第一行：今天，第一杯，甜。

她抬头笑。

许南：都有。你慢慢选。

## 结尾画面

清晨阳光照在蓝色早餐车上，蒸汽从窗口飘出，像一封终于被打开的信。
