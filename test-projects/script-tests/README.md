# Script Test Projects

This folder contains local script-only test projects for future workflow trials.

Current rule:

- Each subfolder is one independent script test.
- These folders are not initialized as `official-ai-video` projects yet.
- Do not run `init`, `sync`, `verify`, or Obsidian export against them until a later task explicitly asks for that.
- The scripts are intentionally different in genre, structure, tone, and visual demands.

## Test Set

- `01-neon-harbor-rain`: near-future sci-fi suspense.
- `02-blue-breakfast-van`: contemporary healing family road drama.
- `03-borrowed-sword-inn`: period martial-arts comedy.
