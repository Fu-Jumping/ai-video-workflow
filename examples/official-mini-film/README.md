# Official Mini Film Example

This is the smallest committed demo project for `ai-video-workflow`.

It demonstrates:

- the six-step AI video workflow structure
- one Step 3 storyboard card
- one Step 4 image prompt file
- one Step 5 video prompt file
- Step 6 execution mapping back to Step 3, Step 4, and Step 5
- Codex runtime files generated from the official pack

## How to Verify

From the repo root:

```powershell
pnpm build
pnpm example:verify
```

## What This Example Does Not Demonstrate Yet

- Obsidian vault export
- LibTV canvas execution
- MCP server runtime
- direct image or video generation
