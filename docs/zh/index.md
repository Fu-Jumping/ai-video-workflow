# ai-video-workflow

`ai-video-workflow` 帮助 AI 视频创作者在不同 IDE、workflow pack 和生成平台之间运行同一套结构化工作流。

- 用 `official-ai-video` 作为默认官方教学 pack。
- 用 CLI 做初始化、同步、校验、诊断和脚手架生成。
- 用文档站理解工作流、平台接入和扩展方式。
